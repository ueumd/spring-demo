Spring 概念
Spring 是开源的 Java EE 的轻量级开发框架，用于简化企业应用开发的复杂度

有两个核心部分：

IOC：控制反转，把创建对象的过程交给 Spring 进行管理
AOP：面向切面，不修改源代码进行功能增强
Spring 的特点
方便解耦，简化开发
AOP 编程支持
方便测试程序
方便集成其它框架
方便进行事务操作
降低 Java EE API 使用难度
